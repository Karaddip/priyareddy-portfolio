# My Portfolio

A modern, responsive portfolio website built with React, TypeScript, and Tailwind CSS, showcasing my projects and professional experience.

## 🚀 Live Demo

Visit my portfolio: [https://polampellys.github.io/Portfolio/](https://polampellys.github.io/Portfolio/)

## 🛠️ Built With

- React
- TypeScript
- Tailwind CSS
- Vite
- Shadcn/ui Components
- React Router

## ✨ Features

- Responsive design that works on all devices
- Dark/Light theme support
- Project showcase with detailed case studies
- Skills and certifications display
- Interactive contact form
- Smooth animations and transitions

## 🔧 Local Development

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/polampellys/Portfolio.git

# Navigate to the project directory
cd Portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

## 📝 Project Structure

```
Portfolio/
├── src/
│   ├── components/     # Reusable UI components
│   ├── pages/         # Page components
│   ├── styles/        # Global styles
│   └── App.tsx        # Main application component
├── public/            # Static assets
└── index.html         # Entry point
```

## 🚀 Deployment

This portfolio is deployed using GitHub Pages. To deploy new changes:

```bash
npm run deploy
```
